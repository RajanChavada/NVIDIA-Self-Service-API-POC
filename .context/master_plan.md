## Goal
Build a robust, event-driven Self-Service Portal for NVIDIA engineers to request ephemeral GPU infrastructure (e.g., "4 H100s for 2h") without writing Terraform/Kubectl manually.

## Core Tech Stack Components

### Backend
- FastAPI
- Asyncio
- SQLAlchemy


## Frontend
- React
- TypeScript
- Material UI
- Vite
- Hosted on Vercel


## Database
- PostgreSQL
- SQLAlchemy + Asyncpg 

## Message Queue
- Kafka AIOkafka
